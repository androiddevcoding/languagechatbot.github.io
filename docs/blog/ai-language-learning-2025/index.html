    • Политика конфиденциальности

    • Language Chatbot
    • January 1, 2026

    Настоящая Политика конфиденциальности описывает порядок сбора, использования, хранения и защиты информации пользователей сайта и приложения Language Chatbot (далее - Сервис).

    Контакт для связи по вопросам обработки персональных данных:
    Email: codingpas@gmail.com

    1. Общие положения

    1.1. Используя Сервис, пользователь подтверждает свое согласие с настоящей Политикой конфиденциальности.

    1.2. В случае несогласия с условиями Политики пользователь обязан прекратить использование Сервиса.

    1.3. Политика применяется ко всей информации, которую Сервис может получить о пользователе в процессе использования сайта и приложения.

    2. Состав собираемых данных

    Сервис может собирать следующие категории информации:

    • данные, предоставляемые пользователем добровольно (например, email при регистрации)
    • технические данные об устройстве, браузере, операционной системе и версии приложения
    • данные о взаимодействии с Сервисом (посещенные страницы, действия, продолжительность сессий)
    • обезличенные аналитические и статистические данные
    • информацию о факте оплаты, статусе подписки или покупке платных функций

    Сервис не собирает и не хранит:

    • номера банковских карт
    • CVC/CVV-коды
    • логины и пароли от платежных систем

    3. Цели обработки информации

    Обработка данных осуществляется исключительно в следующих целях:

    • предоставление и поддержка работоспособности Сервиса
    • регистрация и идентификация пользователей
    • предоставление доступа к платным функциям
    • обработка платежей и подтверждение подписок
    • улучшение функциональности, качества и стабильности Сервиса
    • аналитика использования и статистические исследования
    • обработка запросов и обращений пользователей

    4. Платные функции и платежи

    4.1. Сервис может предоставлять платные функции на основе подписки или разовых покупок.

    4.2. Оплата осуществляется через сторонние платежные системы или магазины приложений.

    4.3. Сервис получает только ограниченную информацию, необходимую для подтверждения оплаты:

    • факт оплаты
    • статус подписки или покупки
    • технические идентификаторы транзакций

    4.4. Возвраты и отмена подписок регулируются правилами соответствующих платежных систем и магазинов приложений.

    5. Аналитика и статистика

    5.1. Для анализа использования Сервиса могут применяться сторонние аналитические инструменты.

    5.2. Аналитические данные носят обезличенный и технический характер и не используются для прямой идентификации личности пользователя.

    5.3. Цель аналитики - улучшение качества сервиса, выявление ошибок и оптимизация пользовательского опыта.

    6. Передача данных третьим лицам

    6.1. Сервис не продает и не передает персональные данные пользователей третьим лицам.

    6.2. Передача информации возможна только в следующих случаях:

    • для обеспечения работы Сервиса (инфраструктура, хостинг, аналитика)
    • для обработки платежей и подтверждения подписок
    • при наличии законных требований со стороны государственных органов

    7. Хранение и защита данных

    7.1. Сервис применяет разумные технические и организационные меры для защиты данных от утраты, изменения и несанкционированного доступа.

    7.2. Данные хранятся не дольше, чем это необходимо для целей их обработки, если более длительный срок хранения не требуется законодательством.

    8. Права пользователя

    Пользователь имеет право:

    • получать информацию о своих персональных данных
    • требовать уточнения, обновления или исправления данных
    • запрашивать удаление данных при отсутствии законных оснований для их хранения
    • отзывать согласие на обработку данных
    • прекратить использование Сервиса в любое время

    Для реализации прав пользователь может обратиться по адресу:
    codingpas@gmail.com

    9. Использование файлов cookie

    9.1. Сайт может использовать файлы cookie и аналогичные технологии для обеспечения корректной работы и аналитики.

    9.2. Пользователь может отключить использование cookie в настройках браузера, однако это может повлиять на корректность работы отдельных функций.

    10. Изменения Политики конфиденциальности

    10.1. Сервис вправе вносить изменения в настоящую Политику без предварительного уведомления пользователя.

    10.2. Актуальная версия Политики всегда публикуется на данной странице.
